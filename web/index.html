About This Demo

This is a 3D ASCII renderer written in C++ and compiled to WebAssembly using Emscripten. It demonstrates:

  • 3D geometry classes (Vector3D, Matrix4x4)
  • 3D object rendering (Cube, Sphere, Pyramid)
  • ASCII-based wireframe rendering
  • 3D transformations (rotation, scaling, translation)
  • Depth buffering for correct occlusion
